# Sound Bite Updater

This project provides a Gradle plugin designed to automate the retrieval and processing of sound bites from the official
Play Sounds page.

## Functionality

1. **Downloads** the latest sound bite files from the Play Sounds source.
2. **Converts** all files to the MP3 format to ensure consistency.
3. **Adjusts volume levels** based on configuration specified on the Play Sounds page.
4. **Outputs** the processed files to a configurable directory.
